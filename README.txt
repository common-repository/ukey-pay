== Description ==

### UKey: The open micropayments infrastructure

UKey enables cashless monetization of low-priced digital assets like your WP content. Tap into a new revenue stream and start selling your creative content for 5 cents or more!

== Installation ==

Starting with UKey consists of just two steps: installing and setting up the plugin.

#### Install UKey from WordPress plugin explorer

1. Visit the plugins page within your dashboard and select "Add New"
1. Search for "UKey" or "UKey Pay"
1. Select and activate UKey from the search result
1. Go to "UKey" menu and enter your Client ID; Retrieve the Client ID from UKey merchant dashboard

#### Install UKey manually

1. Upload the "ukey" folder to the /wp-content/plugins/ folder
1. Activate the UKey plugin through the "Plugins" menu in WordPress
1. Go to "UKey" menu and enter your Client ID; Retrieve the Client ID from UKey merchant dashboard

== Frequently Asked Questions ==

= How do I manage pricing for my posts/content =

Login to your UKey merchant [dashboard](https://dashboard.ukey.co) and go to "Assets" menu to manage pricing.
